# 英语单词复习小程序

一个不需要安装依赖的本地网页应用，直接打开 `index.html` 即可使用。

## 功能

- 添加单词、中文释义和例句
- 批量导入，格式为 `英文,中文释义,例句`
- 复习时先看英文，再显示答案
- 用“不会 / 不熟 / 记得 / 熟练”反馈记忆情况
- 背过但不熟的词会自动标为薄弱词，并在队列中更频繁出现
- 数据保存在浏览器本地 `localStorage`
- 支持导出词库 JSON

## 使用

打开：

```text
/Users/kangyifan/word-memorizer/index.html
```

快捷键：

- 空格：显示答案
- 1：不会
- 2：不熟
- 3：记得
- 4：熟练

## 永久网址部署

这个小程序是静态网页，可以部署到 GitHub Pages、Vercel、Netlify 或 Cloudflare Pages。
部署后，网址由云端托管，不再依赖本机是否开机。
